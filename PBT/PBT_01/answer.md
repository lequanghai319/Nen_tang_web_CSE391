# PHẦN A — KIỂM TRA ĐỌC HIỂU

## Câu A1 — HTTP & Browser
*Nguồn tham chiếu: tuan_1_html5/01_introduction_html_universe.md*

### 1. 5 bước xảy ra khi truy cập shopee.vn:
1. **DNS Lookup:** Trình duyệt tìm địa chỉ IP từ tên miền.
2. **TCP Handshake:** Thiết lập kết nối an toàn với máy chủ.
3. **HTTP Request:** Gửi yêu cầu lấy nội dung trang web.
4. **HTTP Response:** Máy chủ gửi lại file HTML, CSS, JS.
5. **Rendering:** Trình duyệt hiển thị trang web lên màn hình.

### 2. Thông tin trong Tab Network:
Tab Network giúp theo dõi giao tiếp giữa trình duyệt và máy chủ, bao gồm:
* Các file đang được tải về.
* Kích thước và thời gian tải của từng file.
* Trạng thái thành công hay thất bại (**Status Code**).
